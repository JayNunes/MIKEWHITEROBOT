STREAMLABS VIRTUAL CURRENCY EXCHANGE

Role:

Product Manager, UI/UX Designer

Medium:

Website, Mobile App & Chrome Extension

This product enables streamers to create their own virtual currency and be able to buy and sell them on an open market with real world dollars.

I managed the design team to create the UI/UX and the engineering team to prototype then build out the final products on web, mobile as well as a chrome extension.

Below are designs for a general overview of the core features for the mobile application and Chrome Extension. These docs were used to pitch the concept and functionality to Directors for sign-off, as well as to the team for alignment. Once approved, individual user stories for each feature were created and spec'd in greater detail and used for sprint planning.
